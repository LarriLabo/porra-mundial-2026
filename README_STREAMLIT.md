# Porra Mundial 2026 - Pack Streamlit

Este pack contiene una app básica de Streamlit para mostrar el ranking de la porra leyendo un Excel `.xlsx` desde Google Drive.

## Archivos
- `app.py`
- `requirements.txt`
- `historial_puntos.csv`
- `README_STREAMLIT.md`

## Uso
1. Crea un repositorio en GitHub.
2. Sube estos archivos.
3. En Streamlit Community Cloud, selecciona el repo, la rama `main` y el archivo `app.py`.
4. Una vez desplegada la app, pega el enlace compartido de Google Drive de tu Excel.

## Importante
- El Excel debe tener una hoja llamada `Puntos`.
- El fichero de Google Drive debe estar compartido como “Cualquier persona con el enlace”.
